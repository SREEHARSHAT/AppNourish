# NutriCoach - Personal Nutrition Tracking App

NutriCoach is a personalized nutrition tracking and meal suggestion application designed to help users monitor their dietary habits, track essential vitamins and minerals, and receive personalized recommendations for improving their diet.

## Features

- **Ingredient Tracking**: Log all ingredients you currently have at home
- **Meal Suggestions**: Get personalized meal recommendations based on available ingredients
- **Nutrition Tracking**: Monitor intake of essential vitamins and minerals
- **Dietary Analytics**: Visualize your nutrition trends over time
- **Personalized Recommendations**: Get suggestions for dietary improvements
- **Cooking Guidance**: Receive tips and recipes for healthy meals

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
```

2. Navigate to the project directory:
```bash
cd nutrition-app
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm start
```

The application will be accessible at http://localhost:3000

## How It Works

1. **Track Ingredients**: Add all ingredients you currently have
2. **Get Meal Suggestions**: Receive recipes based on available ingredients
3. **Monitor Nutrition**: Track your intake of essential nutrients
4. **Analyze Trends**: View your nutrition analytics over time
5. **Improve Diet**: Get personalized recommendations for dietary improvements

## Technology Stack

- React.js for the user interface
- IndexedDB via Dexie.js for local data storage
- Chart.js for data visualization
- React Router for navigation
